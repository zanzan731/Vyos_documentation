# v5.7 Firewall Rules

This file summarizes the effective rules in `v5.5.conf` with the numbering convention:

- `1`: state/return traffic
- `50` series: segment -> internet egress (eth0)
- `100` series: WAN/public -> DMZ public services
- `200` series: INTERNAL clients (`eth2`) -> allowed services only
- `300` series: DMZ east-west/admin/monitoring
- `400` series: IPV6ONLY clients (`eth3`) -> allowed services only
